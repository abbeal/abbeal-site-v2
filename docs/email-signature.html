Installation Gmail (2 signatures)

1. Crée la signature A — "Abbeal Pro"

  1. Sélectionne tout le bloc "Option A" preview ci-dessus → ⌘C
  2. Gmail → ⚙ Settings → See all settings
  3. Tab General → section Signature → Create new
  4. Nom : Abbeal Pro
  5. ⌘V dans l'éditeur → vérifie le rendu

2. Crée la signature D — "Abbeal Reply"

  1. Reste sur la même page Settings
  2. Clique Create new à nouveau
  3. Nom : Abbeal Reply
  4. Sélectionne tout le bloc "Option D" preview → ⌘C → ⌘V dans le nouvel éditeur

3. Configure les défauts

  1. Section Signature defaults sous l'éditeur :
    • For new emails use : Abbeal Pro
    • On reply/forward use : Abbeal Reply
  2. Coche "Insert signature before quoted text in replies" (mieux visuellement)
  3. Scroll en bas → Save changes

4. Test

  1. Compose un nouvel email à toi-même → vérifie que Abbeal Pro (logo + Calendly) s'affiche
  2. Réponds à un email existant → vérifie que Abbeal Reply (ultra réduit) s'affiche
  3. Tu peux switcher manuellement entre les 2 dans la fenêtre de compose si besoin (icône stylo en bas du compose)

Notes

  • Logo proportions corrigées : 160 × 35 (ratio 4.54:1) — fidèle au fichier source 1511×333
  • Pas de téléphones dans aucune des 2 versions — Calendly est ton CTA principal
  • Pas de manifesto dans aucune des 2 — gain de poids, le manifesto reste sur abbeal.com
